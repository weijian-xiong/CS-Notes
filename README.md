## Python
How to install homebrew / python:<br>
Type this command in the terminal:
```
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)”

brew install python
```
How to start with pip:
```
sudo easy_install pip
```
